Definição de compra e venda de imóveis

(dez/2021)

Estudo de insight realizado em Python para identificar as melhores oportunidades de compra e venda de imóveis de uma empresa fictícia, utilizando uma base de dados com diversas características dos imóveis. Foi utilizado estatística descritiva para auxiliar na definição do preço de compra e venda dos imóveis. A expectativa de lucro seria entre $96.220.810 e $244.216.541.

Ferramentas utilizadas:

  • Python, Pandas, Numpy, Seaborn, scikit-learn;
  • Jupyter Notebook;
  • Estatística Descritiva;
  • Correlação de Pearson.
  • Regressão linear.
